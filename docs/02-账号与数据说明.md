# 账号与数据说明

本地版默认依赖开发种子数据，不再保留生产环境初始化逻辑。

## 默认账号

- 管理员：`admin / admin123`
- 学生：`20230001 / 123456`
- 学生：`20230002 / 123456`
- 学生：`20230003 / 123456`

这些账号在首次启动时由 Flyway 的开发数据脚本导入。

## 导入学生

管理后台支持 `.csv` 和 `.xlsx` 文件导入，推荐字段如下：

- `student_no`
- `display_name`
- `college`
- `class_name`
- `username`
- `password`
- `status`

模板文件：

- [student-import-template.csv](./student-import-template.csv)

说明：

- 新导入学生必须提供初始密码
- 已存在学生再次导入时会更新资料
- `status` 仅支持 `ACTIVE` 或 `DISABLED`

## 本地版精简内容

为了减少本地联调复杂度，下面这些能力已经移除：

- 真实微信授权登录
- 微信账号绑定与重置绑定
- 生产环境配置说明
- bootstrap admin 自动创建逻辑

当前版本统一使用账号密码登录，便于后端、后台和小程序联调。
